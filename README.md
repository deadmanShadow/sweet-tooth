# Sweet Tooth - Modern Cake E-commerce Platform

Sweet Tooth is a high-performance, full-stack e-commerce application built for a premium cake shop experience. It features a sleek, responsive design, a robust backend architecture, and a seamless user journey from product discovery to secure checkout.

## 🚀 Overview

This project showcases a modern approach to building e-commerce solutions using a decoupled architecture. The frontend provides a rich, interactive user experience with smooth animations, while the backend ensures secure data management, role-based access control, and efficient order processing.

---

## ⚙️ Backend Tech Stack

The backend is engineered for scalability, security, and maintainability.

- **Framework**: [NestJS](https://nestjs.com/) - A progressive Node.js framework for building efficient, reliable, and scalable server-side applications.
- **ORM**: [Prisma](https://www.prisma.io/) - A type-safe ORM that simplifies database interactions and migrations.
- **Authentication**: [JWT](https://jwt.io/) & [Passport.js](https://www.passportjs.org/) - Secure, stateless authentication with role-based access control (RBAC).
- **Validation**: [Class-validator](https://github.com/typestack/class-validator) & [Joi](https://joi.dev/) - Rigorous server-side data validation.
- **File Storage**: [Cloudinary](https://cloudinary.com/) - Integrated via Multer for high-performance image hosting and optimization.
- **API Documentation**: [Swagger/OpenAPI](https://swagger.io/) - Interactive API documentation for easy integration.
- **Security**: [Helmet](https://helmetjs.github.io/) & [CORS](https://github.com/expressjs/cors) - Implementation of security best practices.

---

## 💻 Frontend Tech Stack

The frontend is built with a focus on speed, responsiveness, and developer experience.

- **Framework**: [Next.js](https://nextjs.org/) (App Router) - Leveraging Server Components for SEO and optimized performance.
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) - Utilizing the latest utility-first CSS framework for rapid and maintainable UI development.
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) - A collection of high-quality, accessible components built on Radix UI.
- **State Management**: React Context API - Efficiently managing global states for Authentication and Shopping Cart.
- **Animations**: [GSAP](https://gsap.com/) - Delivering smooth, professional-grade animations for a premium feel.
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/) - Consistent and scalable iconography.
- **Form Handling**: [React Hook Form](https://react-hook-form.com/) with [Zod](https://zod.dev/) validation - Ensuring robust client-side data integrity.
- **HTTP Client**: [Axios](https://axios-http.com/) - Standardized API communication with interceptors.
- **Notifications**: [Sonner](https://sonner.emilkowal.ski/) (Toasts) & [SweetAlert2](https://sweetalert2.github.io/) - Intuitive and non-intrusive user feedback.

## ✨ Key Features

### 🍰 Product Experience

- **Dynamic Catalog**: Browse a variety of cakes with real-time availability.
- **Detail View**: Rich product pages with high-quality images, flavor options, and size selections.
- **Smart Search**: Quickly find the perfect treat (implemented in admin and browse views).

### 🛒 Shopping & Checkout

- **Persistent Cart**: Items remain in your cart across sessions.
- **Seamless Checkout**: A streamlined process to finalize orders.
- **Order Tracking**: Users can view their order history and status in real-time.

### 🛡️ Secure Dashboards

- **User Dashboard**: Manage profile settings and track past orders.
- **Admin Command Center**:
  - **Inventory Management**: Create, update, and delete products with image uploads.
  - **Order Management**: Oversee all customer orders and update their fulfillment status.
  - **User Management**: Monitor registered users and manage permissions.

### 📱 Responsive & Accessible

- Fully optimized for mobile, tablet, and desktop devices.
- Adherence to modern accessibility standards for an inclusive user experience.

## 🎨 Design Philosophy

Sweet Tooth is designed with a "Content First" approach, emphasizing the beauty of the products through large imagery and a clean, minimalist aesthetic. The use of a refined color palette (oklch-based) and subtle GSAP transitions creates an atmosphere of luxury and quality.
